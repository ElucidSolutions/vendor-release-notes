Highlights

MAS Automated Welcome Letter (Part of the Make It Easy (MIE) Project)

Update/Enhancement

ORS

An award letter will now automatically be sent to a vendor upon MAS contract award. The contract summary will be attached to the award letter by default as a separate PDF document.

Read the details of this change →

Expand Report Offices to Use Numeric Characters

Update/Enhancement

FSS Online

The “Report Office” field has been changed from alphabetical to alphanumeric in all systems which use report offices and/or contract numbers. Numbers 0-9 can now be used for future contract activities.

Read the details of this change →

Allow ACOs Without Signature Authority to Delegate Modifications to ACOs with Signature Authority

Update/Enhancement

FSS Online

If an ACOs does not have signature authority for a modification, FSS Online will prompt them to reassign the mod to an ACO of their choice, rather than the PCO by default.

Read the details of this change →
